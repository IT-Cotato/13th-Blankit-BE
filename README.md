# 13th-Blankit-BE

Blankit 백엔드 프로젝트입니다.

## 기술 스택

- Java 21
- Spring Boot 4.1.0
- Gradle
- Spring Web
- Spring Data JPA
- Spring Security
- Validation
- Lombok
- springdoc-openapi
- MySQL / H2

## 실행 방법

```bash
./gradlew bootRun
```

## Swagger

```text
http://localhost:8080/swagger-ui/index.html
http://localhost:8080/swagger-ui.html
```
